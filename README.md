# Smart Grid Analyzer ⚡

A modern web application built with **React + TypeScript + Vite** for monitoring, analyzing, and visualizing smart grid and energy-related data in real time.

Live Demo: [Smart Grid Analyzer](https://smartgridanalyzer.netlify.app/)

---

# Features

* ⚡ Real-time smart grid monitoring
* 📊 Interactive analytics dashboard
* 📈 Energy consumption visualization
* 🔍 Data filtering and analysis tools
* 🌙 Responsive modern UI
* ⚛️ Built with React + TypeScript
* ⚡ Fast development with Vite
* 🧹 ESLint integration for code quality

---

# Tech Stack

* **Frontend:** React
* **Language:** TypeScript
* **Build Tool:** Vite
* **Styling:** CSS / Tailwind / Custom UI

# Getting Started

## Prerequisites

Make sure you have installed:

* Node.js (v18 or later recommended)
* npm or yarn

Check versions:

```bash
node -v
npm -v
```

---

# Installation

Clone the repository:

```bash
git clone https://github.com/your-username/smart-grid-analyzer.git
```

Navigate into the project:

```bash
cd smart-grid-analyzer
```

Install dependencies:

```bash
npm install
```

---

# Running the Development Server

Start the app locally:

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:5173
```

---

# Build for Production

Create an optimized production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

# ESLint Configuration

This project includes ESLint support for maintaining clean and scalable code.

To enable stricter type-aware linting:

```js
tseslint.configs.strictTypeChecked
```

Optional React lint plugins:

```bash
npm install eslint-plugin-react-x eslint-plugin-react-dom
```
